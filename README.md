# 🌤️ Weather App – Real-Time Weather with Auto Location

A modern and responsive Weather Web Application that shows real-time weather data using OpenWeather API.  
It automatically detects user location and updates UI dynamically with day/night mode and weather icons.

## ⚙️ How It Works

1. User allows location permission or enters city name.
2. JavaScript sends API request using `fetch()`.
3. API returns weather data in JSON format.
4. Data is processed and displayed dynamically using DOM manipulation.
5. Background and weather icon update automatically based on conditions.
